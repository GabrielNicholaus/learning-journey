I recently completed a practicum report focusing on Software Supply Chain Security.

In this report, I analyze the various security risks that can arise during the software development and distribution process,
from identifying vulnerabilities in the dependencies that applications use, investigating secret leaks, to redesigning more secure CI/CD pipelines.
Through this practicum, I gained first-hand experience in applying the principles of Secure Software Delivery and DevSecOps to improve security throughout 
the Software Development Lifecycle (SDLC).

Disclaimer: All scenarios, configurations, data, and vulnerabilities analyzed in this report are part of academic assignments and are used solely 
for learning purposes as well as simulating the implementation of software security practices.
