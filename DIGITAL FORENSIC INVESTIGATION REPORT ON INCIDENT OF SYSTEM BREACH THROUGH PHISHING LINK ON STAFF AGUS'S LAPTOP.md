Digital Forensic Investigation Report: Windows Phishing & Anti-Forensic 
Incident Reconstruction

Recently, I completed a case study report on Digital Forensic Investigations. This 
report was created in fulfillment of my academic assignment to delve into how 
cybersecurity incident analysis is performed on a Windows environment during 
a phishing attack.

In this project, I simulated the entire incident cycle—from phishing email entry, 
malicious file execution, persistence formation through Registry Run Keys, to 
anti-forensic tactics such as timestomping and log cleansing. Using the NIST SP 
800-86 framework approach, I analyzed various important digital artifacts such 
as Master File Table (MFT), Prefetch, Windows Event Log, and network traffic 
analysis using Wireshark to reconstruct the attack chronology thoroughly.

This report is intended purely for educational and learning purposes. Of course, 
as a learning process, this report is far from perfect and there may be analytical
errors or technical errors in it.
